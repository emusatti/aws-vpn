# Make your own VPN on AWS

Follow the step by step guide on Medium: <link>https://medium.com/@emusatti/how-to-build-your-own-vpn-with-aws-597bf5424bcb</link>

How to:
- Upload the script in the User Data field when launching the EC2 instance
- Assign an Elastic IP to the EC2 instance
- SCP the client.ovpn file to your host machine
- Enjoy!
